FLang

Code with words that make sense!

FLang is a super fun programming language that talks like a human. Learn to code, make stuff happen, and have a blast — no boring stuff!

🚀 Start Learning! 🔍 Explore Tools

Look how cool FLang is! 😎

-- Say hello to everyone!
SET name TO "World"
SET score TO 100

DISP "Hello " + name + "!"

IF score > 50:
    DISP "You are AWESOME!"
ELSE:
    DISP "Keep trying!"

Easy to Read
FLang uses plain words like SET, DISP, and IF so you always know what's happening!
